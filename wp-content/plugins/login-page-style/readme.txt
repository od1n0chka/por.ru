== Description ==

You get freedom change login ( wp-login.php) default style And you also change logo and logo background.

== Installation ==

Step 1. Download the Plugin from  from official WordPress Plugin Directory.

Step 2. Upload it in you wp-content/plugins directory.

Step 3. Activate it from Admin panel.

Step 4. You can see "Login page style" option under Settings menu of your WordPress Admin panel. 

Step 5. User with admin power can change the settings of the plugin.
